🎯 MCQ Upload (JSON)

How to upload MCQs:
1. Ask ChatGPT: "Generate 15 MCQs for [topic] in this JSON format"
2. Copy the JSON
3. Paste below and click Upload

Required JSON format:
[{"q":"Question?","opts":["A. option","B. option","C. option","D. option"],"ans":0,"exp":"Why A is correct","topic":"Topic Name","difficulty":"easy"}]

ans = index of correct option (0=A, 1=B, 2=C, 3=D) · difficulty = easy/medium/hard

⚡ Flashcard Upload (JSON)

How to upload flashcards:
1. Ask ChatGPT: "Generate 20 flashcards for [topic] in this JSON format"
2. Copy the JSON
3. Paste below and click Upload

Required JSON format:
[{"q":"What is X?","a":"X is a data structure that...","topic":"Topic Name"}]
